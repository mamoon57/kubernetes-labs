# Kubernetes labs

YAML examples for core K8s objects — pods, replica sets, deployments. Each folder has manifests plus notes in plain English.

## What's here

| Folder | Topic |
|--------|-------|
| [pods/](pods/) | Pod specs, volumes, init containers |
| [replicasets/](replicasets/) | ReplicaSet with emptyDir, NFS, cloud PVCs |
| [deployment/](deployment/) | Deployment examples |

Good for interview prep or onboarding someone new to K8s.
